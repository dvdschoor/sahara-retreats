# Sahara Retreats

Fast, static Astro website for the Sahara Trek & Movement Adventure by Pilates ByLaar.

## Local development

```sh
npm install
npm run dev
```

Use `npm run check`, `npm run lint`, and `npm run build` before publishing.

The public reservation form prepares an email to `pilatesbylaar@gmail.com`; it does not store visitor data or depend on a form backend.
